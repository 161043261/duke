CDN (内容分发网络) 是一组分布在各地的服务器, 这些服务器存储数据的副本
根据哪些服务器与用户距离最近, 提供快速服务
CDN 被用于传输样式表 和 JavaScript 脚本等静态资源
  1. 使用 CDN 向用户分发静态资源, 降低服务器的请求压力
  2. 缩短传输距离, 减小传输延迟
其他
  • Canvas: 2D 图形 JavaScript API
  • 可缩放矢量图形 SVG
  • WebGL: 3D 图形 JavaScript API
  • <audio< 和 <video< 音频和视频
  • WebRTC, Web Real-Time Communications 服务器/客户端实时通信

mp3, mp4, webm 等格式称为容器格式
定义了媒体文件的元数据, 音频轨道 Audio Tracks 和视频轨道 Video Tracks 的存储结构, 编解码格式等
