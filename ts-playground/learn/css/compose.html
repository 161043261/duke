CSS 是如何工作的

  1. 浏览器请求 HTML 文件
  2. 浏览器加载, 解析 HTML 文件, 将 HTML 文件转换为 DOM 树
    DOM 树是 HTML 文件在内存中的表现形式
    DOM 树: <html> 根节点, <body> 子节点...
  3. 浏览器请求图片, 视频, CSS 样式表, JavaScript 脚本等资源
  4. 浏览器加载, 解析 CSS 文件, 根据选择器的类型 (例如标签, 类, ID 选择器) 将选择器 (哈希) 映射到不同的桶中
  5. 渲染: 浏览器将 CSS 规则应用到 DOM 节点, 布局, 着色
  • px: pixel 像素, 绝对长度单位
    无继承性
  • em: element 元素, 相对长度单位
    相对当前元素的字体大小
    如果当前元素的字体大小为 16px, 则 1em 等于 16px
    有继承性: 子元素的宽度为 2em (16px)
浏览器遇到无法解析的 CSS 时: 直接忽略

DOM 案例

<p>
  Let's use:
  <span>Cascading</span>
  <span>Style</span>
  <span>Sheets</span>
</p>

转换为 DOM 树

P
├─ "Let's use:"
├─ SPAN
|  └─ "Cascading"
├─ SPAN
|  └─ "Style"
└─ SPAN
   └─ "Sheets"
