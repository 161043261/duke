| 原字符 | 等价字符引用 |
| ------ | ------------ |
| <      | &lt;         |
| >      | &gt;         |
| "      | &quot;       |
| '      | &apos;       |
| &      | &amp;        |

| 标签           | 说明                    |
| -------------- | ----------------------- |
| p              | 段落                    |
| h1, h2, ... h6 | 标题                    |
| ul, li         | 无序列表 unordered list |
| ol, li         | 有序列表 ordered list   |
| em             | emphasis 等价于斜体     |
| strong         | 等价于粗体              |
| i              | 斜体 italic             |
| b              | 粗体 bold               |
| u              | 下划线 underline        |

统一资源定位符: URL, Uniform Resource Locator
